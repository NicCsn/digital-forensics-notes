# Digital Forensics Field Notes

A personal, open collection of learning notes, cheatsheets, tools, CTF writeups, and lab projects in the field of digital forensics and cybersecurity. Built for students and enthusiasts who prefer practical, searchable field notes over academic wikis.

## Repository Structure

| Folder | Description | Example Topics |
|--------|-------------|----------------|
| `00-meta/` | Templates and shared resources | Content templates, glossary |
| `01-grundlagen/` | Computing fundamentals | Number systems, encoding, logic |
| `02-betriebssysteme/` | Operating systems | File systems, memory, boot process |
| `03-netzwerke/` | Networking | Protocols, packet analysis, Wireshark |
| `04-forensik-methoden/` | Forensic methodologies | Chain of custody, imaging, analysis workflow |
| `05-it-security/` | IT security | Cryptography, hardening, attack vectors |
| `06-osint/` | Open Source Intelligence | OSINT lifecycle, tools, techniques |
| `07-recht-kriminalistik/` | Law and criminalistics | Digital evidence, legal frameworks |
| `08-ki-forensik/` | AI in forensics | LLMs in analysis, machine learning for DFIR |
| `09-tools/` | Custom scripts and tools | Python/Bash utilities for forensic tasks |
| `10-ctf-writeups/` | CTF writeups | HackTheBox, OverTheWire, misc. challenges |
| `11-labs-projekte/` | Lab reports and case studies | Full projects, practical exercises |
| `assets/` | Images and auxiliary files | Screenshots, diagrams |

## How to Navigate

- **By topic:** Browse the folders above. Each contains notes grouped by subject area.
- **By tag:** Use `INDEX.md` to find content by theme, type, and tags. The index is a manually maintained lookup table.
- **By frontmatter:** Every note includes YAML frontmatter with `tags`, `type`, and `difficulty` — these connect related topics across folders without duplicating files.
