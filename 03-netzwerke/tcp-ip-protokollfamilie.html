Kurzueberblick

Die TCP/IP-Protokollfamilie ist das Fundament des Internets. Als Forensiker musst du jedes Feld in TCP- und IP-Headern lesen koennen — darin stecken Zeitstempel, Routing-Informationen und Angreiferspuren.

OSI-Modell vs. TCP/IP

OSI (7 Schichten)

7. Application
6. Presentation
5. Session
4. Transport
3. Network
2. Data Link
1. Physical

TCP/IP (4 Schichten)

Application
HTTP, DNS, SMTP
Transport
TCP, UDP
Internet
IPv4, IPv6, ICMP
Link
Ethernet, ARP

TCP Three-Way-Handshake

Client

SYN Seq=X

→
Server

SYN-ACK Seq=Y, Ack=X+1

←
Client

ACK Seq=X+1, Ack=Y+1

→
Verbindung steht

Datenuebertragung beginnt

Forensisch: SYN ohne ACK = Port-Scan oder Firewall-Block. Viele SYN von einer IP = Scan-Versuch.
