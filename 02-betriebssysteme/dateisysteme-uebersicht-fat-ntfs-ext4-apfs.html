Dateisysteme im Ueberblick Grundlagen

Kurzueberblick

Ein Dateisystem organisiert, wie Daten auf einem Speichermedium abgelegt, gefunden und verwaltet werden. Jedes hat eigene Metadaten, Zeitstempel und Strukturen — forensisch entscheidend.

Sleuth Kit Befehle 
# Partition-Layout
mmls image.dd

# Dateisystem-Info
fsstat -o 2048 image.dd

# Rekursives File Listing (auch geloeschte!)
fls -r -o 2048 image.dd

# Datei per Inode extrahieren
icat -o 2048 image.dd 5
