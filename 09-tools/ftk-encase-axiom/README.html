Kurzueberblick

FTK, EnCase und AXIOM sind die drei dominierenden kommerziellen Forensik-Suiten. Sie werden von Strafverfolgungsbehoerden, Unternehmen und privaten Forensik-Dienstleistern eingesetzt. Dieses Dokument vergleicht die drei Tools hinsichtlich Features, Lizenzmodell, Preis und Einsatzszenarien.

Vorab: Keines dieser Tools ist "das Beste" — die Wahl haengt vom konkreten Use-Case, Budget und der vorhandenen Infrastruktur ab. In der Praxis setzen viele Labs mehrere Tools parallel ein.

FTK — Forensic Toolkit (Exterro)

FTK ist eine der aeltesten und etabliertesten Forensik-Suiten. Besonders stark ist die Index-basierte Volltextsuche (dtSearch-Engine), die selbst auf mehreren Terabytes innerhalb von Sekunden Ergebnisse liefert.

FTK Imager (kostenlos)

FTK Imager ist das kostenlose Imaging-Tool von Exterro — ein Standardwerkzeug fuer jeden Forensiker.

# FTK Imager — Funktionen (kostenlos)
✅ Physische Images (E01, dd/raw) von Laufwerken und Partitionen
✅ Logische Images (einzelne Dateien/Ordner)
✅ Memory Dump von Live-Systemen (Windows, Linux)
✅ Mount von Images als virtuelles Laufwerk
✅ Export von Files/Hashes (Hash-Liste als CSV)
✅ Preview-Modus (Live-System browsen, ohne Imaging)

FTK Suite (kostenpflichtig)

StaerkenSchwaechen
Index-Suche: Instant-Volltextsuche ueber TB-grosse Images — unschlagbar fuer Keyword-HuntsLangsame UI bei grossen Cases (5+ TB) — FTK 7 hat verbessert, bleibt aber ein Thema
Registry Viewer: Integrierter Registry-Parser mit Timeline und ReportingKeine native Cloud-Unterstuetzung — Cloud-Daten muessen separat extrahiert werden
KFF (Known File Filter): Hash-Datenbanken (NSRL, Project VIC) integriert — sortiert known-good/known-badNur Windows — kein Linux-/macOS-Analyse-Rechner
File Carving: Starkes Carving-Modul (Signature- und Header-basiert)Teuer — Lizenzkosten im oberen Bereich
FTK Stichwort: Der dtSearch-Index macht FTK zur ersten Wahl, wenn in grossen Datenmengen nach Schluesselwoertern, E-Mail-Adressen oder Mustern gesucht werden muss. Kein anderes Tool indiziert und durchsucht so schnell.

EnCase — OpenText EnCase Forensic

EnCase ist der Urvater der digitalen Forensik (Markteinfuehrung 1998) und gilt in vielen Gerichtsbarkeiten als Goldstandard fuer gerichtsfeste Analysen. EnCase verwendet das proprietaere EnCase Evidence File Format (.E01 / .Ex01).

StaerkenSchwaechen
Evidence Processor: Automatisiertes Vorverarbeiten von Images (File Carving, Hash, Index, Thumbnails)Steile Lernkurve: UI ist komplex und wenig intuitiv
EnScript: Proprietaere Skriptsprache — extrem maechtig fuer wiederkehrende Analysen und AutomationKein echter Cloud-Support: Cloud-Extraction muss extern erfolgen
Conditions: Bedingungsbasierte Filter, die live auf das Image angewendet werden (kein Index noetig)Dongle-Zwang: Lizenz laeuft ueber Hardware-Dongle — Aerger bei Dongle-Verlust
Enterprise: EnCase Enterprise ermoeglicht Remote-Collection im gesamten FirmennetzPerformance: Bei grossen Cases (10+ TB) langsamer als FTK
Gerichtsverwertbarkeit: Hoch — weltweit vor Gericht etabliertNur Windows fuer Analyse-Rechner
EnScript-Kompetenz: EnScript ist eine der groessten Staerken, aber auch die groesste Einstiegshuerde. Custom EnScripts zu schreiben erfordert Zeit und Training. Wer EnScript beherrscht, kann nahezu jede Analyse automatisieren.

AXIOM — Magnet AXIOM

AXIOM ist der juengste der drei grossen Suiten und hat sich besonders durch Cloud-Acquisition und die Connections-View (visuelle Verknuepfung von Artefakten) einen Namen gemacht.

StaerkenSchwaechen
Cloud-Acquisition: Kein anderes Tool extrahiert Cloud-Daten so umfassend (Google Takeout, Microsoft 365, iCloud, Social Media)Kein eigener Imager: Image muss mit externem Tool (z.B. FTK Imager) erstellt werden
Connections-View: Visuelle Darstellung von Beziehungen zwischen Dateien, Benutzern, Kommunikation (Graph-basiert)Index-Suche: Schwaechere Volltextsuche als FTK (kein dtSearch)
Modernes UI: Intuitiv, modern, schnelle Einarbeitung — niedrigste Lernkurve der drei ToolsPremium-Preis fuer Cloud-Module: Cloud-Acquisition kostet extra
Mobile-Forensik: Nahtlose Integration von iOS/Android-ExtractionsKein EnScript-Aequivalent: Wenig Spielraum fuer tiefe Custom-Automatisierung
Magnet.AI: KI-gestuetzte Bilderkennung (Nacktheit, Waffen, Drogen, etc.)Nur Windows
AXIOM Stichwort: "Cloud-Forensik". Wenn ein Fall Cloud-Daten (Google, Microsoft 365, Social Media) enthaelt, fuehrt kaum ein Weg an AXIOM vorbei. EnCase und FTK decken diesen Bereich nicht in vergleichbarem Umfang ab.

Open-Source-Alternativen

Nicht jedes Budget erlaubt kommerzielle Forensik-Suiten. Diese Open-Source-Tools decken viele Analyse-Beduerfnisse ab:

ToolTypEinsatzzweck
AutopsyGUI-Forensik-PlattformVollstaendige Case-Analyse (auf Sleuth Kit basierend)
Sleuth Kit (TSK)CLI-Forensik-ToolkitDateisystem-Analyse, File Carving, Timeline
Plaso / log2timelineTimeline-GeneratorSuper-Timeline aus allen verfuegbaren Zeitstempeln erstellen
FTK ImagerImager (Freeware)Forensische Images erstellen, mounten, preview
VolatilityMemory-ForensikAnalyse von RAM-Dumps (Prozesse, Netzwerk, Malware)
KAPECollector + ParserGezielte Artefakt-Sammlung und -Parsing (Windows)
VelociraptorLive-Response + HuntingAgentenbasierte DFIR auf Live-Systemen
Open-Source-Stack: Ein bewaehrter Stack ist: FTK Imager fuer Imaging → Autopsy fuer Case-Management und Analyse → Plaso fuer Timeline → Volatility fuer Memory → KAPE fuer Triage. Dieser Stack kostet nichts und deckt >90% aller forensischen Aufgaben ab.
